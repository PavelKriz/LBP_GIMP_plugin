Detailed Description

CULBP is class that provides implementation of improved LBP method.

in ULBP are taken into account only uniform values - uniform value is value that has in binary form max two transitions between 0 and1 examples: 00011111 - uniform 11000111 - uniform 01000110 - non-uniform 01010101 - non-uniform
