\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_c_u_l_b_p}{C\+U\+L\+BP}} is class that provides implementation of improved L\+BP method. 

in U\+L\+BP are taken into account only uniform values -\/ uniform value is value that has in binary form max two transitions between 0 and1 examples\+: 00011111 -\/ uniform 11000111 -\/ uniform 01000110 -\/ non-\/uniform 01010101 -\/ non-\/uniform
