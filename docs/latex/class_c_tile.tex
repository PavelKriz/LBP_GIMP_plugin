\doxysubsection{Detailed Description}
Class that represent and process image tile. 

Helps manage processing tile. Proper use is call at first \mbox{\hyperlink{class_c_tile}{C\+Tile}} init and then call Load\+And\+Process\+Tile

\char`\"{}\+Borders\char`\"{} in documantation meens this\+: Some algorithms need their neigboorhood (radius) so the tile must be bigger then the processing area. Then the tile need not processed borders which contains pixels from neighbour tiles or if its not possible those borders are mirrod from edges of the tile.
