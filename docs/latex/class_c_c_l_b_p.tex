\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_c_c_l_b_p}{C\+C\+L\+BP}} is class that provides implementation of improved L\+BP method. 

in \mbox{\hyperlink{class_c_l_b_p}{C\+L\+B\+P(completed L\+B\+P)}} are computed three parts(\+S,\+M,\+C)\+:

S(classic L\+B\+P) -\/ for( all neigbour\+Points)\{ S $\vert$= (neigbour\+Point\+\_\+k $>$= mean\+Mp\+Value.\+first) $<$$<$ k\}

M(magnitude -\/ like L\+BP but for every neighbor point looks calculation this\+: for( all neigbour\+Points)\{ M $\vert$= (magnitude\+\_\+k $>$= MM) $<$$<$ k\} \begin{DoxyVerb} MM - means mean magnitude of whole image, magnitude_k = |neigbourPoint_k - center|
\end{DoxyVerb}


C(\+L\+B\+P but on place of center is there mean gray value of whole image (\+C\+I) )\+: for( all neigbour\+Points)\{ C $\vert$= (neigbour\+Point\+\_\+k $>$= CI) $<$$<$ k\}

its recommended to take a closer look on this method first somewhere else, becouse its more complicated then other L\+BP methods
